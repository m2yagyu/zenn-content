# 執筆プロジェクトの方針

## このリポジトリの目的
Zennで技術記事を継続publishする。
看板テーマは「物理から理解する生成AI / 拡散モデル」。
統計力学・微分方程式のバックグラウンドを活かし、実装と物理的な理論の間を埋める。

## 記事の執筆スタイル（毎回守る）
- 冒頭2行で「この記事を読むと何ができるようになるか」を明示する
- 動くコードを先に見せ、理論・数式は後半に置く
- 数式を出す前には必ず日本語で直感的な説明を添える
- 読者はコードは書けるが数学・物理に不安がある層を想定する
- 物理の概念（統計力学、微分方程式など）を持ち出すときは、
  まず身近なアナロジーで説明してから数式に入る

## コードの扱い（信用に直結するので厳守）
- 記事に載せるコードは必ずこのセッション内で実行して動作確認する
- 動作確認が取れていないコードは記事に含めない
- 実装が数式・理論を正しく反映しているか、対応関係をコメントで示す

## 図解
- 拡散過程の可視化、分布の変化、アーキテクチャ図は
  matplotlib / mermaid で生成する

## 公開フロー
- Zennのフロントマター（title, emoji, type, topics, published）を整える
- topics には diffusion, machinelearning, python, 物理 などを想定
- 公開前に一度ローカルプレビュー（npx zenn preview）で確認する
